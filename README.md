# Sistema de Ficha de Cadastro

Projeto simples desenvolvido em Python para realizar o cadastro de usuários através do terminal.

## Funcionalidades

- Cadastro de nome
- Cadastro de idade
- Cadastro de cidade
- Cadastro de profissão
- Cadastro de altura
- Cadastro de e-mail
- Cadastro de hobbies
- Exibição dos dados cadastrados

## Tecnologias Utilizadas

- Python 3

## Como Executar

1. Clone este repositório:

```bash
git clone hhttps://github.com/linconsous/cadastro_python
```

2. Acesse a pasta do projeto:

```bash
cd ficha_cadastro
```

3. Execute o arquivo principal:

```bash
python ficha.py
```

## Exemplo de Uso

```text
Digite seu nome: João Silva
Digite sua idade: 25
Digite seu e-mail: joao@email.com

=== Ficha de Cadastro ===
Nome: João Silva
Idade: 25
E-mail: joao@email.com
```
